# Invoxa — Brand assets

Carpeta con el logotipo en formatos vectoriales (SVG) y rasterizados (PNG).

## SVG (escalables, recomendados)
- `isotipo-cafe.svg` — símbolo en café #2a1f17
- `isotipo-terracota.svg` — símbolo en terracota #c2410c
- `isotipo-crema.svg` — símbolo en crema #fbf7f0 (sobre fondo oscuro)
- `isotipo-contenido-cafe.svg` — símbolo dentro de cuadro café (app icon)
- `imagotipo-horizontal-cafe.svg` — lockup horizontal positivo
- `imagotipo-horizontal-crema.svg` — lockup horizontal negativo
- `imagotipo-horizontal-terracota.svg` — lockup horizontal en acento
- `imagotipo-vertical-cafe.svg` — lockup apilado
- `imagotipo-tagline-cafe.svg` — lockup con tagline
- `wordmark-cafe.svg` / `wordmark-crema.svg` — solo texto

## PNG (carpeta `png/`)
- Isotipo: 256, 512, 1024 px
- Imagotipos: 800, 1600 px de ancho

> Para impresión y producción usa los SVG. Los PNG con wordmark
> usan fallback serif del sistema; si necesitas Fraunces exacta,
> exporta desde el SVG abriéndolo en un navegador con la fuente
> instalada o desde un editor vectorial.
